# LakeCity Transport Inquiry Website

This is a simple Django project based on a transport inquiry system.
Users can fill a form related to transport service and the submitted data
is saved in the database. The admin can view and manage all inquiries
from the Django admin panel.

This project is created for learning Django basics and backend development.

---

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite Database

---

## Project Features
- Transport inquiry form
- Data stored in database
- Admin panel to manage inquiries
- Simple and clean project structure
